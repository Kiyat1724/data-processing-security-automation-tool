---

### 📦 Installation

```bash
python -m venv .venv
```

Activate environment:

```bash
.venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

### ▶️ Execution

```bash
python main.py
```
